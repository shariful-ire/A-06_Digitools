# 🎯 Digital Tools Buying Website

## 📌 Project Overview

Digital Tools Buying Website is a modern and responsive web application where users can explore different digital tools and purchase them easily. Users can add products to the cart, view selected items, and proceed to checkout with a smooth user experience.

## ✨ Features

### 🛍️ 1. Product Listing System

* Displayed 6–10 digital tools in a responsive 3-column layout
* Each product includes name, price, description, features, and tags

---

### 🛒 2. Dynamic Cart System

* Add products to cart
* Remove individual products
* Cart count updates in Navbar
* “Proceed to Checkout” clears all cart items

---

### 🔄 3. Toggle System (Product & Cart)

* Default view shows products
* Toggle buttons to switch between Product and Cart view
* Cart shows empty message initially

---
